# 💳 Personal Finance Behavioral Insights Dashboard

## 📌 Project Overview

This project analyzes personal financial transactions to uncover spending patterns, behavioral trends, and opportunities for savings.

## 🎯 Objective

To transform raw transaction data into meaningful insights that help in better financial decision-making.

## 🛠️ Tools Used

* Power BI
* Python (Pandas, Matplotlib)
* Data Cleaning & EDA

## 📊 Dashboard Features

* KPI metrics (Total Spending, Transactions, Average Spending)
* Top 5 Spending Categories
* Monthly Spending Trend
* Day-wise and Time-wise Spending Analysis
* Interactive filters (Month & Category)

## 🧠 Key Insights

* Majority of spending is concentrated in top categories like Food and Others
* Weekend spending is significantly higher than weekdays
* Peak spending occurs during evening hours
* Spending trends vary across months
* Opportunities exist to reduce non-essential expenses

## 🚀 How to Use

* Download the `.pbix` file
* Open in Power BI Desktop
* Interact with filters and visuals

## 📌 Conclusion

This project demonstrates how data analytics can be used to understand financial behavior and drive better decision-making.
